Perguntas frequentes

A documentação de como criar um formulário pode ser encontrada através do README do projeto no Github. A principio o formulário deve conter os seguintes campos: nome e email, certifique-se de aplicar as validações corretamente.

O ID do formulário é uma chave de indentificação do mesmo e encontrado através de sua URL. Veja o seguinte exemplo: docs.google.com/forms/d/e/1FAIpQLSfCMgP4-UVdcAMrN41yg8ihx4hIJ3eTreiuDFgy68mGL1qseA/viewform

O projeto é open source e você pode contribuir com ele no Github. A comunidade do Gufers tem o objetivo de estimular e divulgar o desenvolvimento de software. É um grupo colaborativo, sem fins lucrativos, que visa conectar pessoas, compartilhar conhecimentos, organizar eventos e treinamentos, motivar indivíduos e organizações a investirem em tecnologia para melhorar os negócios, a sociedade e o mundo.

O Prize Draw usa as respostas do formulario para realizar um sorteio aleatório. Não fazemos nenhum tipo de coleta de dados. Este código é aberto e você pode conferir o que fazemos com seus dados.
